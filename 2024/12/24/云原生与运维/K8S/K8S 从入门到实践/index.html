ж–°зҡ„ж–№ејҸжҳҜйҖҡиҝҮйғЁзҪІе®№еҷЁж–№ејҸе®һзҺ°пјҢжҜҸдёӘе®№еҷЁд№Ӣй—ҙдә’зӣёйҡ”зҰ»пјҢжҜҸдёӘе®№еҷЁжңүиҮӘе·ұзҡ„ж–Үд»¶зі»з»ҹпјҢе®№еҷЁд№Ӣй—ҙиҝӣзЁӢдёҚдјҡзӣёдә’еҪұе“ҚпјҢиғҪеҢәеҲҶи®Ўз®—иө„жәҗгҖӮзӣёеҜ№дәҺиҷҡжӢҹжңәпјҢе®№еҷЁиғҪеҝ«йҖҹйғЁзҪІпјҢеӣ дёәе®№еҷЁдёҺеә•еұӮи®ҫж–ҪгҖҒжңәеҷЁж–Үд»¶зі»з»ҹжҳҜи§ЈиҖҰзҡ„


Kubernetes жҳҜдёҖдёӘиҪ»дҫҝзҡ„е’ҢеҸҜжү©еұ•зҡ„ејҖжәҗе№іеҸ°пјҢз”ЁдәҺз®ЎзҗҶе®№еҷЁеҢ–еә”з”Ёе’ҢжңҚеҠЎгҖӮйҖҡиҝҮ Kubernetes иғҪеӨҹиҝӣиЎҢеә”з”Ёзҡ„иҮӘеҠЁеҢ–йғЁзҪІе’Ңжү©зј©е®№гҖӮеңЁ Kubernetes дёӯпјҢдјҡе°Ҷз»„жҲҗеә”з”Ёзҡ„е®№еҷЁз»„еҗҲжҲҗдёҖдёӘйҖ»иҫ‘еҚ•е…ғд»Ҙжӣҙжҳ“з®ЎзҗҶе’ҢеҸ‘зҺ°

Kubernetes з§ҜзҙҜдәҶдҪңдёә Google з”ҹдә§зҺҜеўғиҝҗиЎҢе·ҘдҪңиҙҹиҪҪ 15 е№ҙзҡ„з»ҸйӘҢпјҢе№¶еҗёж”¶дәҶжқҘиҮӘдәҺзӨҫеҢәзҡ„жңҖдҪіжғіжі•е’Ңе®һи·ө

зү№жҖ§дёҺеҠҹиғҪ

  • иҮӘеҠЁиЈ…з®ұ

    еҹәдәҺе®№еҷЁеҜ№еә”з”ЁиҝҗиЎҢзҺҜеўғзҡ„иө„жәҗй…ҚзҪ®иҰҒжұӮиҮӘеҠЁйғЁзҪІеә”з”Ёе®№еҷЁ

  • иҮӘжҲ‘дҝ®еӨҚ

    еҪ“е®№еҷЁеӨұиҙҘж—¶пјҢдјҡеҜ№е®№еҷЁиҝӣиЎҢйҮҚеҗҜ

    еҪ“жүҖйғЁзҪІзҡ„ Node иҠӮзӮ№жңүй—®йўҳж—¶пјҢдјҡеҜ№е®№еҷЁиҝӣиЎҢйҮҚж–°йғЁзҪІе’ҢйҮҚж–°и°ғеәҰ

    еҪ“е®№еҷЁжңӘйҖҡиҝҮзӣ‘жҺ§жЈҖжҹҘж—¶пјҢдјҡе…ій—ӯжӯӨе®№еҷЁзӣҙеҲ°е®№еҷЁжӯЈеёёиҝҗиЎҢж—¶пјҢжүҚдјҡеҜ№еӨ–жҸҗдҫӣжңҚеҠЎ

  • ж°ҙе№іжү©еұ•

    йҖҡиҝҮз®ҖеҚ•зҡ„е‘Ҫд»ӨгҖҒз”ЁжҲ·UI з•ҢйқўжҲ–еҹәдәҺ CPU зӯүиө„жәҗдҪҝз”Ёжғ…еҶөпјҢеҜ№еә”з”Ёе®№еҷЁиҝӣиЎҢи§„жЁЎжү©еӨ§жҲ–и§„жЁЎеүӘиЈҒ

  • жңҚеҠЎеҸ‘зҺ°

    з”ЁжҲ·дёҚйңҖдҪҝз”ЁйўқеӨ–зҡ„жңҚеҠЎеҸ‘зҺ°жңәеҲ¶пјҢе°ұиғҪеӨҹеҹәдәҺ Kubernetes иҮӘиә«иғҪеҠӣе®һзҺ°жңҚеҠЎеҸ‘зҺ°е’ҢиҙҹиҪҪеқҮиЎЎ

  • ж»ҡеҠЁжӣҙж–°

    еҸҜд»Ҙж №жҚ®еә”з”Ёзҡ„еҸҳеҢ–пјҢеҜ№еә”з”Ёе®№еҷЁиҝҗиЎҢзҡ„еә”з”ЁпјҢиҝӣиЎҢдёҖж¬ЎжҖ§жҲ–жү№йҮҸејҸжӣҙж–°

  • зүҲжң¬еӣһйҖҖ

    еҸҜд»Ҙж №жҚ®еә”з”ЁйғЁзҪІжғ…еҶөпјҢеҜ№еә”з”Ёе®№еҷЁиҝҗиЎҢзҡ„еә”з”ЁпјҢиҝӣиЎҢеҺҶеҸІзүҲжң¬еҚіж—¶еӣһйҖҖ

  • еҜҶй’ҘдёҺй…ҚзҪ®з®ЎзҗҶ

    еңЁдёҚйңҖиҰҒйҮҚж–°жһ„е»әй•ңеғҸзҡ„жғ…еҶөдёӢпјҢеҸҜд»ҘйғЁзҪІе’Ңжӣҙж–°еҜҶй’Ҙе’Ңеә”з”Ёй…ҚзҪ®пјҢзұ»дјјзғӯйғЁзҪІ

  • еӯҳеӮЁзј–жҺ’

    иҮӘеҠЁе®һзҺ°еӯҳеӮЁзі»з»ҹжҢӮиҪҪеҸҠеә”з”ЁпјҢзү№еҲ«еҜ№жңүзҠ¶жҖҒеә”з”Ёе®һзҺ°ж•°жҚ®жҢҒд№…еҢ–йқһеёёйҮҚиҰҒ

    еӯҳеӮЁзі»з»ҹеҸҜд»ҘжқҘиҮӘдәҺжң¬ең°зӣ®еҪ•гҖҒзҪ‘з»ңеӯҳеӮЁ(NFSгҖҒGlusterгҖҒCeph зӯү)гҖҒе…¬е…ұдә‘еӯҳеӮЁжңҚеҠЎ

  • жү№еӨ„зҗҶ

    жҸҗдҫӣдёҖж¬ЎжҖ§д»»еҠЎпјҢе®ҡж—¶д»»еҠЎпјӣж»Ўи¶іжү№йҮҸж•°жҚ®еӨ„зҗҶе’ҢеҲҶжһҗзҡ„еңәжҷҜ

    • controller-managerпјҡеӨ„зҗҶйӣҶзҫӨдёӯеёёи§„еҗҺеҸ°д»»еҠЎпјҢдёҖдёӘиө„жәҗеҜ№еә”дёҖдёӘжҺ§еҲ¶еҷЁ
    • etcdпјҡеӯҳеӮЁзі»з»ҹпјҢз”ЁдәҺдҝқеӯҳйӣҶзҫӨдёӯзҡ„зӣёе…іж•°жҚ®

  • е®№еҷЁиҝҗиЎҢзҺҜеўғгҖҗContainer RuntimeгҖ‘
    • е®№еҷЁиҝҗиЎҢзҺҜеўғжҳҜиҙҹиҙЈиҝҗиЎҢе®№еҷЁзҡ„иҪҜд»¶
    • Kubernetes ж”ҜжҢҒеӨҡдёӘе®№еҷЁиҝҗиЎҢзҺҜеўғпјҡDockerгҖҒcontainerdгҖҒcri-oгҖҒrktlet д»ҘеҸҠд»»дҪ•е®һзҺ° Kubernetes CRI (е®№еҷЁиҝҗиЎҢзҺҜеўғжҺҘеҸЈ) зҡ„иҪҜд»¶

  • Pod жҳҜеӨҡиҝӣзЁӢи®ҫи®ЎпјҢиҝҗз”ЁеӨҡдёӘеә”з”ЁзЁӢеәҸпјҢд№ҹе°ұжҳҜдёҖдёӘ Pod йҮҢйқўжңүеӨҡдёӘе®№еҷЁпјҢиҖҢдёҖдёӘе®№еҷЁйҮҢйқўиҝҗиЎҢдёҖдёӘеә”з”ЁзЁӢеәҸ

Pod жҳҜеңЁ K8S йӣҶзҫӨдёӯиҝҗиЎҢйғЁзҪІеә”з”ЁжҲ–жңҚеҠЎзҡ„жңҖе°ҸеҚ•е…ғпјҢе®ғжҳҜеҸҜд»Ҙж”ҜжҢҒеӨҡе®№еҷЁзҡ„гҖӮPod зҡ„и®ҫи®ЎзҗҶеҝөжҳҜж”ҜжҢҒеӨҡдёӘе®№еҷЁеңЁдёҖдёӘ Pod дёӯе…ұдә«зҪ‘з»ңең°еқҖе’Ңж–Үд»¶зі»з»ҹпјҢеҸҜд»ҘйҖҡиҝҮиҝӣзЁӢй—ҙйҖҡдҝЎе’Ңж–Үд»¶е…ұдә«иҝҷз§Қз®ҖеҚ•й«ҳж•Ҳзҡ„ж–№ејҸз»„еҗҲе®ҢжҲҗжңҚеҠЎгҖӮеҗҢж—¶ Pod еҜ№еӨҡе®№еҷЁзҡ„ж”ҜжҢҒжҳҜ K8S дёӯжңҖеҹәзЎҖзҡ„и®ҫи®ЎзҗҶеҝөгҖӮеңЁз”ҹдә§зҺҜеўғдёӯпјҢйҖҡеёёжҳҜз”ұдёҚеҗҢзҡ„еӣўйҳҹеҗ„иҮӘејҖеҸ‘жһ„е»әиҮӘе·ұзҡ„е®№еҷЁй•ңеғҸпјҢеңЁйғЁзҪІзҡ„ж—¶еҖҷз»„еҗҲжҲҗдёҖдёӘеҫ®жңҚеҠЎеҜ№еӨ–жҸҗдҫӣжңҚеҠЎ

Pod жҳҜ K8S йӣҶзҫӨдёӯжүҖжңүдёҡеҠЎзұ»еһӢзҡ„еҹәзЎҖпјҢеҸҜд»ҘжҠҠ Pod зңӢдҪңиҝҗиЎҢеңЁ K8S йӣҶзҫӨдёҠзҡ„е°ҸжңәеҷЁдәәпјҢдёҚеҗҢзұ»еһӢзҡ„дёҡеҠЎе°ұйңҖиҰҒдёҚеҗҢзұ»еһӢзҡ„е°ҸжңәеҷЁдәәеҺ»жү§иЎҢгҖӮзӣ®еүҚ K8S зҡ„дёҡеҠЎдё»иҰҒеҸҜд»ҘеҲҶдёәд»ҘдёӢеҮ з§Қ

  • й•ҝжңҹдјәжңҚеһӢпјҡlong-running
  • жү№еӨ„зҗҶеһӢпјҡbatch
  • иҠӮзӮ№еҗҺеҸ°ж”Ҝж’‘еһӢпјҡnode-daemon
  • жңүзҠ¶жҖҒеә”з”ЁеһӢпјҡstateful application

дёҠиҝ°зҡ„еҮ з§Қзұ»еһӢпјҢеҲҶеҲ«еҜ№еә”зҡ„е°ҸжңәеҷЁдәәжҺ§еҲ¶еҷЁдёәпјҡDeploymentгҖҒJobгҖҒDaemonSet е’Ң StatefulSet

еӣ дёә Pod дёӯеҢ…еҗ«дәҶеҫҲеӨҡдёӘе®№еҷЁпјҢеҒҮи®ҫжҹҗдёӘе®№еҷЁеҮәзҺ°й—®йўҳдәҶпјҢйӮЈд№Ҳе°ұдјҡи§ҰеҸ‘ Pod йҮҚеҗҜжңәеҲ¶

  • AlwaysпјҡеҪ“е®№еҷЁз»ҲжӯўйҖҖеҮәеҗҺпјҢжҖ»жҳҜйҮҚеҗҜе®№еҷЁпјҢй»ҳи®Өзӯ–з•Ҙ гҖҗnginx зӯүпјҢйңҖиҰҒдёҚж–ӯжҸҗдҫӣжңҚеҠЎгҖ‘
  • OnFailureпјҡеҪ“е®№еҷЁејӮеёёйҖҖеҮәпјҲйҖҖеҮәзҠ¶жҖҒз Ғйқһ 0пјүж—¶пјҢжүҚйҮҚеҗҜе®№еҷЁ
  • NeverпјҡеҪ“е®№еҷЁз»ҲжӯўйҖҖеҮәпјҢд»ҺдёҚйҮҚеҗҜе®№еҷЁ гҖҗжү№йҮҸд»»еҠЎгҖ‘

жңүзҡ„ж—¶еҖҷпјҢзЁӢеәҸеҸҜиғҪеҮәзҺ°дәҶ Java е ҶеҶ…еӯҳжәўеҮәпјҢзЁӢеәҸиҝҳеңЁиҝҗиЎҢпјҢдҪҶжҳҜдёҚиғҪеҜ№еӨ–жҸҗдҫӣжңҚеҠЎдәҶпјҢиҝҷдёӘж—¶еҖҷе°ұдёҚиғҪйҖҡиҝҮ е®№еҷЁжЈҖжҹҘжқҘеҲӨж–ӯжңҚеҠЎжҳҜеҗҰеҸҜз”ЁдәҶпјҢиҝҷдёӘж—¶еҖҷе°ұеҸҜд»ҘдҪҝз”Ёеә”з”ЁеұӮйқўзҡ„жЈҖжҹҘ

  • иҠӮзӮ№йҖүжӢ©еҷЁж Үзӯҫ

    жҲ‘д»¬еҸҜд»ҘйҖҡиҝҮд»ҘдёӢе‘Ҫд»ӨпјҢз»ҷжҲ‘д»¬зҡ„иҠӮзӮ№ж–°еўһж ҮзӯҫпјҢз„¶еҗҺиҠӮзӮ№йҖүжӢ©еҷЁе°ұдјҡиҝӣиЎҢи°ғеәҰдәҶ

  • иҠӮзӮ№дәІе’ҢжҖ§

    зЎ¬дәІе’ҢжҖ§пјҡзәҰжқҹжқЎд»¶еҝ…йЎ»ж»Ўи¶і

Pod жҳҜйҖҡиҝҮ Controller е®һзҺ°еә”з”Ёзҡ„иҝҗз»ҙпјҢжҜ”еҰӮеј№жҖ§дјёзј©пјҢж»ҡеҠЁеҚҮзә§зӯү

Pod е’Ң Controller д№Ӣй—ҙжҳҜйҖҡиҝҮ label ж ҮзӯҫжқҘе»әз«Ӣе…ізі»пјҢеҗҢж—¶ Controller еҸҲиў«з§°дёәжҺ§еҲ¶еҷЁе·ҘдҪңиҙҹиҪҪ

Deployment жҺ§еҲ¶еҷЁ

  • Deployment жҺ§еҲ¶еҷЁеҸҜд»ҘйғЁзҪІж— зҠ¶жҖҒеә”з”Ё
  • з®ЎзҗҶ Pod е’Ң ReplicaSet
  • йғЁзҪІпјҢж»ҡеҠЁеҚҮзә§зӯүеҠҹиғҪ
  • еә”з”ЁеңәжҷҜпјҡweb жңҚеҠЎпјҢеҫ®жңҚеҠЎ

Deployment иЎЁзӨәз”ЁжҲ·еҜ№ K8S йӣҶзҫӨзҡ„дёҖж¬Ўжӣҙж–°ж“ҚдҪңгҖӮDeployment жҳҜдёҖдёӘжҜ” RS(Replica Set, RS) еә”з”ЁжЁЎеһӢжӣҙе№ҝзҡ„ API еҜ№иұЎпјҢеҸҜд»ҘжҳҜеҲӣе»әдёҖдёӘж–°зҡ„жңҚеҠЎпјҢжӣҙж–°дёҖдёӘж–°зҡ„жңҚеҠЎпјҢд№ҹеҸҜд»ҘжҳҜж»ҡеҠЁеҚҮзә§дёҖдёӘжңҚеҠЎгҖӮж»ҡеҠЁеҚҮзә§дёҖдёӘжңҚеҠЎпјҢе®һйҷ…жҳҜеҲӣе»әдёҖдёӘж–°зҡ„ RSпјҢз„¶еҗҺйҖҗжёҗе°Ҷж–° RS дёӯеүҜжң¬ж•°еўһеҠ еҲ°зҗҶжғізҠ¶жҖҒпјҢе°Ҷж—§ RS дёӯзҡ„еүҜжң¬ж•°еҮҸе°‘еҲ° 0 зҡ„еӨҚеҗҲж“ҚдҪң

иҝҷж ·дёҖдёӘеӨҚеҗҲж“ҚдҪңз”ЁдёҖдёӘ RS жҳҜдёҚеҘҪжҸҸиҝ°зҡ„пјҢжүҖд»Ҙз”ЁдёҖдёӘжӣҙйҖҡз”Ёзҡ„ Deployment жқҘжҸҸиҝ°гҖӮд»Ҙ K8S зҡ„еҸ‘еұ•ж–№еҗ‘пјҢжңӘжқҘеҜ№жүҖжңүй•ҝжңҹдјәжңҚеһӢзҡ„дёҡеҠЎзҡ„з®ЎзҗҶпјҢйғҪдјҡйҖҡиҝҮ Deployment жқҘз®ЎзҗҶ

Deployment жҺ§еҲ¶еҷЁйғЁзҪІеә”з”Ё
