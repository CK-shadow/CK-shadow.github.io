更新文档

在 MongoDB 中无论使用客户端 API 还是使用 Spring Data，更新返回结果一定是受影响行数。如果没有数据需要进行更新则返回 0

  • updateFirst() 只更新满足条件的第一条记录
  • updateMulti() 更新所有满足条件的记录
  • upsert() 没有满足更新条件的数据则新增
